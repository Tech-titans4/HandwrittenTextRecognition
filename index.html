How Textract Works

  • Image Upload: Users upload an image of handwritten text through the website.

  • Preprocessing: The image is resized, converted to grayscale, and normalized to prepare it for recognition.

  • Text Recognition: The trained model processes the image to extract text using advanced machine learning techniques.

  • File Generation: The recognized text is converted into a downloadable DOC or PDF file based on the user’s choice.

  • Output Delivery: The file is made available for the user to download directly from the website.
